# Bundled fonts

The app bundles Latin-subset static instances of two typefaces to match the web
app's design system on the Skia targets (iOS/web) and Android:

- **Inter** — © The Inter Project Authors. SIL Open Font License 1.1.
  https://github.com/rsms/inter
- **Epilogue** — © The Epilogue Project Authors. SIL Open Font License 1.1.
  https://github.com/etunni/Epilogue
- **Noto Color Emoji** (food glyphs) — © Google. SIL Open Font License 1.1.
  https://github.com/googlefonts/noto-emoji

Each is subset to Latin + common punctuation and (for Inter/Epilogue) instanced
to specific weights to keep the bundle small. The OFL permits bundling and
redistribution; see each project for the full license text.
